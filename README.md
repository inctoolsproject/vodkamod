Tutorial
------
Recomended for python 2.7

First :
    
    $ apt update && apt upgrade -y
    $ pkg install python2 -y
    $ pkg install git -y

Using git :

    $ git clone http://github.com/bamzky/vodkamod

Python packages :

    $ pip2 install thrift==0.9.3
    $ pip2 install requests
    $ pip2 install rsa

Run Vodka :

    $ cd vodkamod
    $ python2 vbot.py
